What is retrophies ?

Retrophies is an open source working prototype of a Nintendo emulator, running Super Mario Bros, but with trophies/achievements implemented on top of the unhacked ROM.

Can Retrophies be extended to other games or emulators ?

Yes, this page is only a prove of concept about achievements on classic games. This emulator (based on JSNES ) already supports different ROMs from Nintendo; other emulators might support this feature as long as they have access to the real-time memory or if they support Game Genie codes already. Retrophies is also scalable to support multiple games and different trophies per game, although it's a pretty complex manual process so it might take some time to do.

How does Retrophies work ?

Achievements are achieved by looking at specific parts of the game's memory in real time. In a similar way of how cheat cartridges work, like Game Genie. An effective but complex process, as all the achievements have been reverse engineered by looking at the memory, one by one.
